Usefull local development environment
---

- 開発を補助する環境をローカルにVagrantで構築しよう
- ローカルで開発するのに必要な環境
  - ex) Web app用のMySQL
  - ex) 検索app用のElasticsearch
  - ex) 自appから呼び出す他appのモック
- 用意しておくといざという時に便利
  - ex) ログ解析のためのembulk+Kibana4
  - ex) 性能試験用のJmeter, locust
- 作った環境をVagrantfileの形で共有する
